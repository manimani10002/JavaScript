## 🚀 JavaScript 기본 개념 다지기 저장소 (TIL)

---

이곳은 **JavaScript 기본 개념**을 탄탄하게 학습하고 매일 정리 및 기록하기 위한 **TIL (Today I Learned) 레포지토리**입니다.

실제 개발에 치우쳐 미처 깊이 있게 다루지 못했던 핵심 원리들을 보충하고, **자바스크립트의 기초를 견고히** 다지는 것을 목표로 합니다.

### ✨ 주요 목표

- **핵심 개념** 이해 (원시 타입, 객체, 실행 컨텍스트, 스코프, 클로저 등)
- **기본 문법** 및 **데이터 구조** 숙달
- **브라우저 API** 활용 (LocalStorage, DOM 조작)
- **학습 내용**을 체계적으로 정리하여 지식 내재화
